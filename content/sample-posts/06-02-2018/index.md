# Shit Hot Tips on VSCode I wish I'd known earlier

## html

1. Emmet abbreviations. Holy smokes these are great and built into VSCode, for the full [emmet syntax click here](https://docs.emmet.io/abbreviations/syntax/)

eg.
```html
// typing 
.container>.item*10

// will give you 
<div class="container">
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
</div>

```
or

```html
// typing
ul>li.item$*5

// will give you
<ul>
    <li class="item1"></li>
    <li class="item2"></li>
    <li class="item3"></li>
    <li class="item4"></li>
    <li class="item5"></li>
</ul>
```

or

```html
// typing
a{Click me}

// will print out
<a href="">Click me</a>
```
